## Project Overview

**Duration**: 2021.11 — 2022.05  
**Type**: Independent Fieldwork Project  
**Methods**: Questionnaire survey, in-depth interviews, participant observation, social media data mining

## Core Work

- Designed user voice collection protocol, built web scraper capturing **10K+ user feedback posts**
- Established emotional evolution monitoring model and demand prioritization matrix
- Conducted **20+ in-depth interviews** targeting **6 user personas** (age/occupation/digital literacy stratification)
- 2 cities (Chengdu, Xi'an), 5+ commercial/institutional sites, 6-month longitudinal tracking
- Independently produced **40,000+ word research report**

## Key Insights

- **Mental model mismatch**: The same product carries vastly different social meanings and identity constructs across user segments
- User compliance and implicit resistance coexist, and this mindset is not fully linearly correlated with technical operational ability
- High adoption metrics can mask true rejection; build **transparency layers** that expose system logic
